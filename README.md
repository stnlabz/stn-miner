# STN-Labs Miner
In Go

## Notice
This miner was designed to use **MAX** cores on whatever platform it gets launched on.

## About
With a Bitcoin Client look and feel (Command Line / Shell Only)
 - Hashrate
 - Shares Attempted `A`
 - Shares Confirmed `C`
 
 Designed to work with the STN-Labz Stratum Server which interfaces with the Stn-Labz Threat Block Chain.
  - This creates and maintains a typical blockchain loop.
